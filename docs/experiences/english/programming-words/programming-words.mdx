| ------------- | :-----------: |

| emphasize     |   اهمیت دادن، تایید کردن   |
| compilation     |   تلفیقی   |
| enhance     |   تقویت کردن   |
| rather     |   بلکه   |
| demonstrate     |   نشان دادن   |
| retrieve     |   پس گرفتن, بازیابی   |
| negligible     |   ناچیز, جزئي   |
| applicable     |   مناسب, خوب   |
| leverage     |   قدرت نفوذ, اهرم   |
| Disclaimer     |   سلب مسئولیت   |
| in contrast     |   متقابلا، در مقابل   |
| approach     |   رویکرد   |
| absence      |   فقدان   |
| reduction       |   کاهش   |
| majority        |   اکثریت   |
| consider         |   در نظر گرفتن   |
| ambitious         |   بلند پروازانه، جاه طلب   |
| bottleneck          |   تنگنا   |
| leveraging           |   اهرم کردن   |
