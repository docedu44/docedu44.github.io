FTP در webstorm#

برای وصل شدن به FTP در webstorm کافیه Alt + Ctrl + S رو بزنیم بعد کلمه SFTP رو سرچ کنیم

بعد وارد قسمت Deployment می شیم و بعد گزینه + رو می زنیم و سپس SFTP رو میزنیم و یک اسم برای سرور انتخاب می کنیم

حالا توو صفحه جدید باید گزینه SSH configuration را انتخاب می کنیم و دوباره + رو میزنیم و بعد باید username و مابقی اطلاعات مربوط به سرور و اتصال به آن رو وارد میکنیم.

حالا هر وقت بخوایم می تونیم فولدر مورد نظر رو انتخاب کنیم و deployment رو بزنیم و upload to server رو بزنیم.
