### ساختار های فعل go

1. به همراه حرف اضافه `to`

👈 بعد از آن، مکان مورد نظر قرار می گیرد:

``` text
We will go to England next week.
```

2. به همراه حرف اضافه `on`

🚩 به معنای `ادامه دادن`

👈 بعد از آن، مصدر `ing`دار قرار می گیرد:

``` text
They went on studing until the morning.
```

🚩 به معنای `رفتن`

👈 بعد از آن، اسم قرار می گیرد:

``` text
We will go on a vacation next week.
```

3. بدون هیچ حرف اضافه ای

👈 بعد از آن، مصدر `ing`دار قرار می گیرد:

``` text
I go walking every day.
```

### صحبت در مورد تصمیمات آینده

1. بیان تصمیمات از قبل گرفته شده:

✨ ساختار `be going to`

``` text
I'm going to fly to Russia on Wednesday.
```

2. بیان قرار ها و برنامه های آینده

✨ زمان حال استمراری

``` text
I'm flying to Russia on Wednesday.
```

3. بیان پیش بینی بر اساس شواهد فعلی:

✨ ساختار `be going to`

``` text
The sun is going down.It's going to be dark.
```

### ساختار have و has

به معنای **داشتن** و **خوردن** می باشد.

🎉 فعل `has`:
1. فاعل های سوم شخص مفرد
2. ضمائر فاعلی `he, she, it`

🎉 فعل `have`:
1. فاعل های غیر سوم شخص مفرد
2. ضمائر فاعلی `i, we, you, they`

``` text
He never has dinner late.
```

✨ می توانیم برای بیان مالکیت از `have got` و `has got` نیز استفاده کنیم

``` text
You have got a house.
```

روش منفی کردن `have`و `has`:
1. با قرار دادن `don't` و `doesn't`قبل از آن
2. تبدیل `has` به `have`

``` text
She doesn't have a big house.
```

روش منفی کردن `have got`و `has got`:
1. با قرار دادن `not` بعد از `have` یا `has`

``` text
She hasn't got a book.
```

روش سوالی کردن `have`و `has`:
1. با قرار دادن `do` و `does`در ابتدای جمله
2. تبدیل `has` به `have`

``` text
Does he have an expensive car?
```

روش سوالی کردن `have got`و `has got`:
1. با جابه جایی فاعل با `have / has`

``` text
Have they got many books?
```
