### چگونه nest را استارت بزنیم؟

می توان به وسیله دستورات زیر اولین پروژه nestjs را بسازیم:

<div dir="ltr">

```cmd
1st step:
    npm i -g @nestjs/cli
2st step:
    nest new project-name
```

</div>

### چگونه یک module بسازیم؟

می توان به وسیله دستور زیر یک ماژول بسازیم که به صورت خودکار به ماژول ها اضافه می شود:

<div dir="ltr">

```cmd
    nest g module module-name
```

</div>
