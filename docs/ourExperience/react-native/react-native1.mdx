### تگ ها

`View`:
معادل همان `div` می باشد که برای مشخص کردن هر بخش از آن استفاده میکنیم.

`Text`:
معادل همان `p` می باشد که برای قرار دادن متن هایمان از آن استفاده می کنیم.

`TouchableOpacity`
 در واقع توسط این تگ می توان المان هایی ماننده `Image` و `Text` را `touchable` کنیم و از رویداد هایی ماننده `onPress` برای آن ها استفاده کنیم.

### Tips

🎉 در `React Native` در واقع `flex` به صورت پیش فرض تمام عرض و ارتفاع باقی مانده را به خود اختصاص می دهد و می توان درون آن دوباره از `flex` برای فرزندان نیز استفاده کرد:


<div dir="ltr">

```jsx
    return (
    <View style={{flex :1 }}>

        <View style={{flex :1, backgroundColor: '#FF5722'}} />

        <View style={{flex :2, backgroundColor: '#673AB7'}} />

        <View style={{flex :3, backgroundColor: '#FFEB3B'}} />

    </View>
    );
```

</div>
